## 🔧 Customization

### Update Contact Info
Edit `components/sections/Contact.tsx`:
```tsx
const contactMethods = [
  { link: 'mailto:YOUR_EMAIL' },
  { link: 'https://wa.me/YOUR_PHONE' },
  // ... etc
]
```

### Add/Remove Projects
Edit `components/sections/Projects.tsx`:
```tsx
const projects = [
  // Add your projects here with metrics
]
```

### Change Colors
Edit `tailwind.config.js`:
```javascript
colors: {
  'charcoal': '#121212',
  'amber-gold': '#F59E0B',
}
```

## 🚀 Deployment Options

### Option 1: Vercel (Recommended)
```bash
npm run build
vercel deploy
```

### Option 2: Netlify
- Connect GitHub repo
- Build command: `npm run build`
- Publish directory: `.next`

### Option 3: Docker
```dockerfile
FROM node:18-alpine
WORKDIR /app
COPY . .
RUN npm install && npm run build
EXPOSE 3000
CMD npm start
```

### Option 4: Traditional Hosting
```bash
npm run build
npm run start
```
Deploy the `.next` folder and `public` folder
